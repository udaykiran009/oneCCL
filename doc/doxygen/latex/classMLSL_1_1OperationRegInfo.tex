\subsection{Detailed Description}
A class to hold \hyperlink{classMLSL_1_1Operation}{Operation} registration information. 

Holds the information about learnable parameters and activation shapes and is used for creation of an \hyperlink{classMLSL_1_1Operation}{Operation} object. All the input/output activation shapes and parameter shapes (if any) should be added before calling \hyperlink{classMLSL_1_1Session_a165a77b678517fd7991777974681916c}{Session\-::\-Add\-Operation()}
